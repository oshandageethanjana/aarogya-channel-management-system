# Aarogya Channel Management System 🏥

This is a web-based Channel Management System developed as part of my HNDIT 2nd Year project. The system is designed to streamline the appointment booking process for patients and provide an efficient management interface for healthcare providers.

## 🛠️ Technologies Used
* **Frontend:** HTML, CSS, JavaScript
* **Backend:** PHP
* **Database:** MySQL (XAMPP environment)

## ✨ Key Features
* **Patient Module:** Easy online appointment scheduling and management.
* **Doctor Module:** Searchable doctor profiles and specialized service listings.
* **Admin Dashboard:** Comprehensive control panel to manage appointments, doctor records, and user inquiries.
* **Secure Database:** Structured MySQL database for reliable data storage and retrieval.
